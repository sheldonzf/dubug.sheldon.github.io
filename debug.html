node 源码调试

代码配置步骤

  1. 下载 node 源码 和 CLion 客户端
  2. sh configure 执行配置文件 (1hours)
  3. make -C out BUILDTYPE=Debug -j 4 根据 makefile 的配置，进行编译，生成可执行文件
  4. make install 安装 (0.5hours)

IDE 配置

  1. 导入项目(New Project)
  2. executable ⇒ out/Debug/node
  3. program arguments ⇒ test.js 新建调试脚本
  4. working directory ⇒ projectPath 工作目录脚本
  5. Before launch 删除 build
